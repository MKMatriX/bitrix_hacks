# Набор моих инструментов для битрикса.

В некоторых подпапках есть свои readme.md.

## components - примеры компонентов.
Абстрактного - empty.
callback - коллбек.
profile.detail - деталка пользователя-покупателя. Управление полями пользователя и профилями покупок через аяксы
profile.enter - Регистрация, Логин, Забыли пароль, Смена забытого пароля

## js. Чуть подчищенный дубликат из модуля mkmatrix.main

## modules - примеры модулей.
Абстрактный - company_name.module_name
mkmatrix.main - мой обычный
mkmatrix.export - модуль экспорта и импорта товаров в корзине в csv и xls. Требует "phpoffice/phpspreadsheet".
mkmatrix.iblockwebp - вместо resizeImageGet использовать код места, содержащий размер, формат изображения.
mkmatrix.sku - еще одна попытка улучшить импорт из 1С, делая торговые предложения из свойства с XML_ID родителя.

## scripts - папка со скриптами для локалки, подключать ***ИСКЛЮЧИТЕЛЬНО*** на локалке. Я подключал через docker-compose. И не переносить бекапы с локалки, если эта папка подключена.

.gitignore_example - решал проблемы, когда папки local еще не было, и далеко не все ее использовали.
.rsync_example - и конечно любимый мною rsync ибо использовать гит для битрикса возможно только если заставить всех сеошников использовать гит.
