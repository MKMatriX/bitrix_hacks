# скрипты

Обычно я их подключал через композер на все локальные проекты. Правда я всегда помнил что перенос через бекап скопирует скрипты позволяющие зайти под админом.

## groups
Переключение групп пользователя. Если нужно тестировать поведение нескольких не групп пользователей, причем все они не администраторы, то весьма помогает.

## login
На сайт ведь нужно зайти, причем в админку. В общем нужно зайти под своим пользователем, которого возможно нужно еще и создать. Этот скрипт для этого.

## hlBake
Заранее извиняюсь за вид кода. Причины были. Но штука классная. Делает модели из хайлоадов. Т.е. у нас есть и модель которую мы можем использовать даже не подключая модуль хайлоадов. И при этом все это еще работает через админку. Впрочем стоит помнить о всяких подводных камнях типа эвентов, или того что если нужны множественные свойства, то придется модуль хайлоадов уже подключать. Но для всякой мелочевки, типа слайдера баннеров - отличная штука.
