# Импорт и экспорт корзины.

## Зачем?
Обычно не нужно, если мы покупает меньше десятка наименований. Однако в b2b сегменте могут покупать сотни наименований. Поэтому им удобней экспортировать это в эксельку и отредактировать там, чем лазить по всему каталогу. Вот эту проблему и решает этот модуль.

Сам модуль требует модуль mkmatrix.main для работы. Хотя конечно это больше для работы избранного. Так что можно выпилить. Правда придется выпиливать из js тоже.

В папке `catalog.component.template` то что можно закинуть в комплексный компонент каталога, если заодно хочется страничку где все эти товары ищутся и выводятся. Предполагается что по урлу /catalog/opt/ будет открывать форма массовой закупки по артикулу. Сам артикул как обычно идет из 1С в свойстве `CML2_ARTICLE`
