Это компонент формы обратной связи, который я копирую из проекта в проект допиливая что-то. Как результат он напоминает снежный ком. Подключается в стиле:

```php
$APPLICATION->IncludeComponent(
	"mkmatrix:callback",
	".default",
	[
		"FIELDS" => [
			// тут список полей, тип поля определяет по коду, и вид в файлике custom_fields.
			[
				"CODE" => "MESSAGE",
				"RU_NAME" => "Сообщение"
			],
			[
				"CODE" => "EMAIL_TO",
				"RU_NAME" => "Имейл"
			],
			[
				"CODE" => "PHONE",
				"RU_NAME" => "Телефон"
			],
		],
		"THEME" => "Форма обратной связи", // тема письма
		"TEMPLATE_NAME" => "callback..default", // наименование шаблона письма.
		"NAME" => "callback",
		"CACHE_TIME" => 36000000,
		"CACHE_TYPE" => "A"
	],
	false
);
```

Примечателен компонент тем, что автоматически создает шаблон письма, что освобождает от набора всякой фигни в админке. А еще он дублирует все в веб-формы. Не знаю зачем, просто делает.

Для его работы по идее нужны js модули mk_ajax_forms, mk_callback. Они отвечают за аякс работу всей формы. Их код в папке js.
